Contrast

Eddyfi Technologies

eddyfi.com/en

I love how Eddyfi Technologies designed its web page in the contrast topic principle. The web page shows an image carousel in the second plane while the first plane puts text and buttons that are visible.

Alignment

Handshake

joinhandshake.com.me

Today Handshake is one of my favorite web pages. This time I take time to look carefully. The page uses the Alignment principle between the mobile view and pc view. was fabulous how the main image, text, label, and buttons are re-ordered.

Proximity

Rollingstone

rollingstone.com

When I need to remember one song and its age I'm going to Rollingstone web page. I choose to look at the Proximity principle on this page. I found how the relationship between items on this page conveys a message and purpose to the user.
